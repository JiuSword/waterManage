### 启动项目

#### :cherry_blossom:后端启动
```bash
cd server/water-order-backend #定位项目
npm install #安装依赖
npm start #启动
```

#### :seedling:前端启动

```bash
yarn #安装依赖
yarn start #启动
```

备注：其他脚本请自行阅读package.json 中的 script
